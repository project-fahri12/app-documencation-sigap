Instalasi

Salin perintah berikut di terminal Anda untuk melakukan setup awal proyek:

// Clone dan Install
git clone https://github.com/sigap/core.git
composer install && npm install

// Setup Environment
cp .env.example .env
php artisan key:generate
php artisan migrate --seed

Role & Hak Akses

Role Wewenang Utama
Admin Manajemen master data, setting periode PPDB, dan laporan akhir.
Panitia Verifikasi dokumen fisik, input nilai tes, dan wawancara.
Calon Siswa Melengkapi profil, upload berkas, dan monitoring status.
