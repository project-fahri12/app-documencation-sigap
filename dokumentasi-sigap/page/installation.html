2. Langkah-Langkah Instalasi

Langkah 1: Clone & Setup
# Masuk ke folder htdocs/www
git clone https://github.com/fahridwi/sigap.git
cd sigap

# Install dependensi backend
composer install
Langkah 2: Environment & Key
cp .env.example .env
php artisan key:generate

Buka file .env dan sesuaikan konfigurasi DB_DATABASE, DB_USERNAME, dan DB_PASSWORD sesuai local server Anda.

Langkah 3: Integrasi Library Wajib

SIGAP menggunakan library khusus untuk pembayaran dan laporan. Jalankan perintah ini:

3. Finalisasi Database

Pastikan Anda sudah membuat database kosong bernama sigap_db di phpMyAdmin sebelum menjalankan perintah di bawah.
# Jalankan migrasi dan isi data awal
php artisan migrate --seed

# Aktifkan akses folder upload
php artisan storage:link

# Jalankan frontend compiler (Vite)
npm install && npm run dev
