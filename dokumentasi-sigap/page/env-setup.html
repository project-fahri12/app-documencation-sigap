Konfigurasi Environment

Mengatur file .env untuk menghubungkan core system, payment gateway, dan sistem laporan.

4. Pengaturan File .env

Salin file .env.example menjadi .env, kemudian sesuaikan parameter kunci di bawah ini:

Koneksi Database (Arsitektur) 
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=sigap_db
DB_USERNAME=root
DB_PASSWORD=
Kosongkan DB_PASSWORD jika menggunakan XAMPP/Laragon bawaan.
Payment Gateway

Integrasi Midtrans oleh Fahri Dwi H.

MIDTRANS_SERVER_KEY=SB-Mid-xxx
MIDTRANS_CLIENT_KEY=SB-Mid-xxx
MIDTRANS_IS_PRODUCTION=false

Ambil key di Midtrans Dashboard

Laporan & Berkas

Manajemen Laporan oleh Saprudi

APP_URL=http://127.0.0.1:8000
FILESYSTEM_DISK=public
APP_TIMEZONE=Asia/Jakarta

APP_URL wajib sesuai agar gambar di PDF muncul.
