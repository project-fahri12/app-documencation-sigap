Panduan Administrator

Instruksi lengkap pengelolaan sistem SIGAP, mulai dari konfigurasi data master hingga penarikan laporan pendaftaran.

1. Ringkasan Dashboard

Setelah login sebagai admin, Anda akan melihat statistik pendaftaran secara real-time. Dashboard ini memberikan gambaran cepat mengenai jumlah pendaftar, status pembayaran, dan verifikasi dokumen.

Tampilan Utama Dashboard Admin SIGAP

2. Pengelolaan Data Master

Data master adalah pondasi sistem. Admin wajib mengatur data ini sebelum periode pendaftaran dibuka.

Tahun Ajaran/Gelombang

Mengatur periode aktif pendaftaran dan kuota masing-masing jalur pendaftaran.

Kategori & Biaya

Setting komponen biaya pendaftaran yang akan terhubung secara otomatis ke Midtrans.

3. Verifikasi & Sistem Laporan

Bagian krusial untuk validasi data pendaftar dan mencetak laporan akhir.

Alur Verifikasi Dokumen
  • Cek kelengkapan berkas (KK, Ijazah, Foto) pada menu Data Pendaftar.
  • Klik tombol Validasi untuk menyetujui data.
  • Sistem akan otomatis mengirim notifikasi/status ke dashboard siswa.
Export Laporan (Saprudi Feature)

Admin dapat menarik seluruh data pendaftaran dalam format Excel (.xlsx) atau ringkasan PDF berdasarkan rentang tanggal atau gelombang.
